# Proyecto Colaborativo en Python

## Descripción
Este repositorio fue creado para desarrollar un proyecto colaborativo en Python utilizando GitHub. El objetivo es que los miembros del equipo trabajen juntos en diferentes módulos del proyecto y utilicen herramientas de control de versiones como ramas y pull requests.

## Estructura del proyecto

El proyecto está dividido en tres módulos principales:

- **main.py**
  Archivo principal desde donde se ejecuta el programa y se llaman los diferentes módulos.

- **matematicas.py**
  Contiene funciones para realizar operaciones matemáticas básicas como suma, resta, multiplicación y división.

- **cuento.py**
  Contiene una función que imprime un cuento corto en pantalla.

## Cómo usar el programa

1. Clonar el repositorio:
git clone https://github.com/Roryd2126/proyecto_colaborativo_python.git

2. Entrar a la carpeta del proyecto:
cd proyecto_colaborativo_python

3. Ejecutar el archivo principal:
python main.py

## Colaboración

Cada miembro del equipo debe:

- Crear una rama para trabajar en sus cambios.
- Realizar modificaciones en el código.
- Enviar un **Pull Request** para integrar los cambios al repositorio principal.
- Revisar el código cada compañeros antes de aprobar los cambios.

Esto permite trabajar de forma organizada y mantener el control de versiones del proyecto.
